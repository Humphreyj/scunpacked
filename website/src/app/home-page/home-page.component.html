Welcome to Star Citizen速 Unpacked!

This site aims to provide an easy way to browse and compare the various ships, vehicles, equipment, weapons and armour available in the game. I am still working on identifying and extracting all the necessary ship and item statistics, so if you find some stat is not listed please feel free to add an issue in GitHub and I'll do my best to add it.

Ultimately there will also be a public API to make it easy for other websites to consume the data without having to go through the process of extracting the data from the game files first, but while I'm building the site, the data in the API is frequently changing, so it might be a bit frustrating to try and use it at the moment.

If you would like to contribute, comment or complain then please visit the GitHub project page and open an issue.

Latest updates

  1. Website
    Insurance times
    Insurance times and costs have been added to the ship information and ship comparison pages.
  2. 3.12.0k PTU
    Notable changes
    This is a very small patch. The improved hull and nacelle strength of the Constellation series has been walked-back a bit and the name of the Ares was changed from "Ares Starfighter" to "Ares Star Fighter".
  3. 3.12.0j PTU
    Notable changes

    The Constellation series has received hull buffs. All critical sections of the ship have had their health buffed by a factor of 2 or even 4! It will be much harder to detach the nacelles too. Acceleration been has improved on all thrusters, but this has come at a cost: the SCunpacked estimated time till the tanks run empty has dropped from 7,600 seconds to just 2,961. Tweaks have also been made to "ground turbulence" and "precision max distance & landing multiplier", whatever they are :)

    The Terrapin thrusters have been improved, the maneuvering thrusters in particular. Fuel capacity has been doubled to compensate for the increased fuel consumption.

    The maneuvering thrusters on the Herald have been substantially buffed. No matching increase in fuel tank capacity though.

    The stock loadouts on the Talon and Shrike have been updated with different armour, coolers, powerplants, shields and quantum drive. The thrusters have received cooling improvements.

    The retro and maneuvering thrusters on the Prospector have been buffed. No improvements to tank capacity or intake rate means the time to empty has dropped from 2,600 seconds to 2,100 seconds.

    The Apocalypse Arms Predator Scattergun has been added with 3466 physical damage per second.

    The MaxOx NDB-30 Repeater has been added with 532 energy damage per second.

    The size 1 Banu shield (also called Sukoran) has been added. 6000 health, 270 regeneration.

  4. Website
    Radars, scanners and pings
    The website now includes basic information about radars, scanners and pings. It looks like there is a lot of detail to decode, so more information will be added later.
  5. API
    Cargo capacity corrections
    It turns out that some cargo grids are flagged as "MiningOnly", meaning that they can't be used to store actual cargo.
  6. API
    Loadout improvements

    In cases where a ship's didn't fully specify the loadout of a component with sub-components, the loadout of the component itself is now used. This means that the cluster munitions in Rattler II missiles and a few strangely configured cargo grids are now correctly loaded.
